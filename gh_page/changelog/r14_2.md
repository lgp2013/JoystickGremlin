# Release 14.2

## New Features
- Show error message if Gremlin can't start due to system misconfiguration
- Show information and warnings about incorrect action or action sequence configurations
- Keyboard input tab added
- Visualization of action sequences on input buttons

## Fixes
- Better handling of vJoy setups
- Improve QML object cleanup
- Device swap improvements
- Disable input highlighting when using modal windows
- Improve handling of invalid actions
- Device bar properly scrolls when more devices than space are present
- Calibrations no longer required to be centered around 0
- Fix logical device inputs not updating properly
- Action ordering now lives in the Actions tab of the options screen
- Correctly handle paths with special characters
